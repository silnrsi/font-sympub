FONTLOG for the SymPub fonts

This file provides detailed information on the SymPub Font Software.
This information should be distributed along with the SymPub fonts and any derivative works.

Basic Font Information

SymPub is an open font with characters that help with typesetting projects.
Font sources are in UFO3 format.
