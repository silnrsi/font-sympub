# SymPub [![Build Status](https://build.palaso.org/app/rest/builds/buildType:Fonts_SymPub/statusIcon)](https://build.palaso.org/viewType.html?buildTypeId=Fonts_SymPub&guest=1)

SymPub (Symbols for Publishing) is an open font with characters that help with typesetting projects. These characters are in three categories:

- Footnote callers - A wide selection of characters already sized and raised for footnote (and other) callers
- Bottom brackets - Several brackets that can be used to mark textual variants, such as in a glossary.

The distribution package includes TTF, WOFF, and WOFF2 versions (see the web folder).

## Encoding details

For a full list of all the symbols available see [Encoded Symbols](documentation/encoding.md).

## Font development processes

The font sources are in UFO3 format. The build process requires [smith](https://github.com/silnrsi/smith) and project build parameters are set in the [wscript](wscript).
